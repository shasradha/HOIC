# HOIC (High Orbit Ion Cannon)

## Overview
HOIC (High Orbit Ion Cannon) is a network stress testing tool that can be used for testing servers under heavy loads. It is a more powerful version of LOIC (Low Orbit Ion Cannon), and should only be used for ethical and authorized stress testing. Unauthorized use is illegal and unethical.

## Installation

### Steps to Install:
1. **Download the HOIC archive**:
   - Go to the [releases page](https://github.com/shasradha/HOIC) and download the `.rar` file.

2. **Extract the file**:
   - Right-click the `.zip` file and select "Extract All..." or use any unzip tool to extract the contents to a folder of your choice.

3. **Run HOIC**:
   - After extracting, navigate to the folder where the files were extracted.
   - Simply **double-click** the `HOIC.exe` file to launch the program.
   - The application is pre-configured and ready to use.

4. **Start testing**:
   - Once HOIC is running, input the target server's IP or domain name, set your desired parameters, and click the "Start" button to begin the stress test.

## Notes:
- **No additional installation required**: HOIC is pre-configured and all necessary files are included.
- **Run as administrator**: Depending on your system, you might need to run HOIC as an administrator for full functionality.
- **Legal Disclaimer**: Ensure that you have permission from the server owner before conducting any testing. Unauthorized use of HOIC for attacks is illegal.
